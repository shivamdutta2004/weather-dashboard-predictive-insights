# 🌦️ Weather Dashboard

A simple web app to check real-time weather and forecasts.

## 🚀 Features

✔ Search by city
✔ Use current location 📍
✔ Temperature, humidity, wind
✔ 5-day forecast 📅
✔ AQI 🌫️ and map 🗺️

## 🛠️ Tech

HTML • CSS • JavaScript • OpenWeather API

## ▶️ Run

Open in VS Code → Run with Live Server

## 🔑 API Key

Add in `script.js`:

const API_KEY = "YOUR_API_KEY";
